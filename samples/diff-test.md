# Diff Test

比較方法:

1. [diff-test-before.ts](/media/mokume1shine/Lake/github/nucle-noct/samples/diff-test-before.ts) と [diff-test-after.ts](/media/mokume1shine/Lake/github/nucle-noct/samples/diff-test-after.ts) を比較する
2. [diff-test-before.json](/media/mokume1shine/Lake/github/nucle-noct/samples/diff-test-before.json) と [diff-test-after.json](/media/mokume1shine/Lake/github/nucle-noct/samples/diff-test-after.json) を比較する

確認ポイント:

- 追加行の背景と文字色
- 削除行の背景と文字色
- 変更行の強調
- 文字列・数値・プロパティ名が diff 上でも読めるか
- JSON の構造記号が diff 上でも埋もれないか
